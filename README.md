# Gestión de Gastos

Aplicación de escritorio para la gestión y control de gastos personales desarrollada en JavaFX.

## Descripción

La aplicación permite registrar gastos personales, organizarlos en categorías, visualizarlos en tablas y gráficos, filtrarlos, configurar alertas de gasto, gestionar cuentas compartidas e importar datos desde ficheros externos.

## Cómo ejecutar

```bash
mvn clean javafx:run
```
